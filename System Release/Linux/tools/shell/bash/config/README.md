# 配置

## Bash 开启 vi-mode

> [Bash 和 Zsh 开启 vi-mode](https://www.bbsmax.com/A/QV5Zv8Ob5y/)

```shell
set -o vi # 临时开启 vi 模式
vi ~/.bashrc # 在其中写入 `set -o vi`
source ~/.bashrc # 刷新配置
```
