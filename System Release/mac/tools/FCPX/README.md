### 浏览器

> 媒体整理，包括分角色、关键词设定和粗剪

| 快捷键 | 说明               | 备注               |
| ------ | ------------------ | ------------------ |
| j      | 倒放               | 按几下就是乘几倍   |
| k      | 暂停               |                    |
| l      | 播放               | 按几下就是快进几倍 |
| k+j    | 逐帧倒放           |                    |
| k+l    | 逐帧播放           |                    |
| i      | 视频片段入点       |                    |
| o      | 视频片段出点       |                    |
| f      | 将视频放入个人收藏 |                    |
| u      | 重做               |                    |
| ;      | 上一个编辑点       |                    |
| ,      | 下一个编辑点       |                    |
