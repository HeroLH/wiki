## 简介

Axios是一个开源的可以用在浏览器端和 `Node JS` 的异步通信框架， 她的主要作用就是实现 AJAX 异步通信，其功能特点如下：

- 从浏览器中创建`XMLHttpRequests`
- 从node.js创建http请求
- 支持Promise API[JS中链式编程]
- 拦截请求和响应
- 转换请求数据和响应数据
- 取消请求
- 自动转换JSON数据
- 客户端支持防御XSRF(跨站请求伪造)
